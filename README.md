# Linux Backup Automation System
**Developed by Juliano Soares**

## 📋 About the Project
This is a professional Bash automation tool for secure backups.

## 🛠️ Tech Features
* **Integrity:** SHA256 Checksum validation.
* **Scheduling:** Fully compatible with Crontab.
* **Logging:** Real-time monitoring with rotating logs.
* **Retention:** Automatic 7-day backup rotation.
